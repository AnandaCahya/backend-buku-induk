## Jalankan

Copy project

```bash
  git https://github.com/bwx1y/backend-buku-induk.git
```

Pergi ke directory

```bash
  cd backend-buku-induk
```

Install dependencies

```bash
  npm install
```

jalankan server

```bash
  npm start
```
